### Step 4: Register Domain (Optional but Recommended)

**Option A: Use .io domain (Premium)**
- Cost: ~$25-50/year
- Registrars: Namecheap, GoDaddy, etc.

Choose one and proceed.

### Step 5: Enable GitHub Pages (5 minutes)

1. Go to your GitHub repo
2. **Settings** → **Pages**
3. Under **Build and deployment**
4. Select **Source: GitHub Actions**
5. This auto-deploys your frontend!

If using custom domain (Option A):
6. Add custom domain: `drawkaro.io`
7. Update DNS at registrar (see DOMAIN_SETUP.md)
